Token-efficient orientation. For the *why*, read [docs/ARCHITECTURE.md](../ARCHITECTURE.md);

## Core logic (read these)

| Area | Path | Entry | Notes |
|------|------|-------|-------|
| Pure simulation | `packages/engine-core/src/` | `index.ts` | No DOM/node/vendor/`Math.random`. Folds events into `World`. |
| · determinism | `…/time/` | `rng.ts`, `clock.ts` | seeded sfc32 RNG + fixed-timestep clock |
| · state | `…/state/` | `world.ts` | flat-JSON `World`, `snapshot.ts` hashing |
| · events | `…/events/` | `apply.ts` | `applyEvent` is the only state mutator; `effects.ts`, `log.ts` |
| · systems | `…/systems/` | per-file | movement, interaction, quests, combat, dialogue, reactions, bribe |
| · conditions | `…/conditions/` | `conditions.ts` | safe expression language (no eval) |
| · ports | `…/ports/` | renderer/narrative/audio | interfaces the app injects |
| Content treaty | `packages/content-schema/src/` | `index.ts` | Zod schemas: `ids`, `npc`, `quest`, `location`, `faction`, `item`, `effect`, `condition`, `assertion`, `pack` |
| Load + validate | `packages/content-loader/src/` | `index.ts` | `load.ts`, `integrity.ts`, `registries.ts`, `canon-graph.ts`, `hash.ts` |
| Story runtime | `packages/narrative-ink/src/` | `index.ts` | the ONLY `inkjs` importer |
| Rendering | `packages/render-pixi/src/` | `index.ts` | the ONLY `pixi.js` importer |
| Save / replay | `packages/persistence/src/` | `index.ts` | idb-keyval store + JSON export |
| Composition root | `packages/app-web/src/` | `main.ts` | wires engine + render + persistence; `session.ts`, `scene.ts`, `input.ts`, `hud.ts`, `beats.ts`, `dialogue-controller.ts` |
| Offline pipeline (NOT shipped) | `tools/pipeline/src/` | `index.ts` | `llm/`, `prompts/`, `schemas/`, `pipelines/`, `synthesis.ts`, `bundle.ts`, `canon.ts`, `scaffold.ts`, `bake.ts` |
| CLIs | `tools/scripts/*.ts` | — | the `pnpm` script bodies (schema-export, content-new, compile-ink, content-validate, content-verify, pipeline-*) |
| Migrations | `tools/migrate/src/` | `index.ts` | `runner.ts`, `world.ts` |

## Tests
- **Unit:** colocated `*.test.ts` beside source in `packages/*/src/` and `tools/*/src/`.
- **Browser e2e:** `packages/app-web/test/*.spec.ts` (Playwright; non-blocking in CI).
- **Determinism gate:** `packages/engine-core/src/state/replay.test.ts` (`pnpm replay:verify`).

## Config
`package.json` (scripts) · `pnpm-workspace.yaml` · `tsconfig.json` (pure) + `tsconfig.dom.json` (DOM) + `tsconfig.base.json` · `eslint.config.js` · `.dependency-cruiser.cjs` (vendor isolation) · `vitest.config.ts` · `.prettierrc.json` · `packages/app-web/playwright.config.ts` · CI `.github/workflows/verify.yml`.

## Skip paths (token-heavy / generated)
See `.aiignore`. Notably: `node_modules/`, `dist/`, `coverage/`, `schema/generated/` (`pnpm schema:export`), `content/canon-index.json` (`pnpm pipeline:export`), `pnpm-lock.yaml`. The `compiled` field inside each `content/**/pack.json` is large generated Ink JSON (regenerated by `pnpm content:compile-ink`) — read the surrounding fields, not it.
